# bass-of-denial-cad
These are the files for the bass I'm creating for Hack Club's Stardance! As someone that loves music, I thought building a bass would be really cool, and since I also like anime, I wanted it to look like Ryo Yamada's Fender Precision from Bocchi the Rock. But because that was somehow still too boring to me, I decided to also make it so the bass can react to overplayed guitar songs, the main inspiration being the 'No stairway! Denied!' scene in Wayne's World.

This repo contains screenshots, STL files, and STEP files for the bass, which was fully CADed in Onshape. It also contains the wiring diagram, which I found on StewMac (the manufacturers of the pickups I'm ordering)'s website, as well as my bill of materials.
